# Soup Simulator Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Measure delivery-ratio and airtime-cost vs node density for the polleneus pure-flooding "uniform soup", with enough methodological rigor that the located percolation cliff is real, not an artifact.

**Architecture:** A deterministic, time-stepped discrete simulation. Nodes move (static-uniform primary, RWP overlay), meet over a unit-disk radio with analytically-timed contacts, and blindly exchange fixed-size blobs under a density-aware airtime budget, a bounded buffer, and a sliding-window seen-record. A metrics layer holds the only knowledge of sender/recipient (the scoring oracle); the engine/policies are addressing-blind. A scenario runner sweeps density with replications and reports mean delivery + 95% CI plus a logistic cliff fit.

**Tech Stack:** Python 3.11+, numpy (geometry, `default_rng`, `SeedSequence`). matplotlib optional (import-guarded). pytest. No SimPy.

## Global Constraints

- Python **3.11+**; dependencies limited to **numpy** (core) and **matplotlib** (optional, import-guarded). Test with **pytest**.
- **Determinism:** one injected `numpy.random.Generator` threaded explicitly; **no module-global `np.random.*` or `random.seed`** anywhere (lint-enforced). Substreams via `SeedSequence.spawn`.
- **Invariant guard (inv 2 pure-flooding, inv 4 sealed-sender):** `model`/`engine`/`policies` code may read **only** `{id, created_at, ttl, size}` of a blob. `sender`/`recipient` live only in the metrics oracle (lint-enforced).
- **Absolute TTL** (parent §6): a blob expires at `created_at + ttl` everywhere; no node extends it.
- **Eviction** (parent §9.5): evict from the **oldest-by-creation** cohort with randomized tie-break (retain younger); **never** closest-to-TTL.
- **Every reported number is an UPPER BOUND** on real-world delivery; the README must say so.
- All sim code under `sim/`. Package `soup_sim/`; tests `sim/tests/`.

```python
# sim/soup_sim/config.py
from dataclasses import dataclass, asdict
import numpy as np

@dataclass(frozen=True)
class Config:
    n:int; width:float; height:float; radius:float; boundary:str; mobility:str
    speed_min:float; speed_max:float; dt:float; ttl:float; buffer_cap:int
    throughput_ideal:float; alpha:float; t_setup:float; p_fail:float; blob_size:float
    warmup:float; measure_window:float; drain:float; n_messages:int
    seen_margin:float; master_seed:int

    def validate(self) -> None:
        if self.boundary not in ("torus", "walls"): raise ValueError("boundary")
        if self.mobility not in ("static", "rwp"): raise ValueError("mobility")
        if self.speed_min < 0 or self.speed_max < self.speed_min: raise ValueError("speed")
        if self.mobility == "rwp" and self.speed_min <= 0: raise ValueError("rwp needs speed_min>0")
        if self.speed_max * self.dt > self.radius / 4.0:
            raise ValueError(f"CFL: speed_max*dt={self.speed_max*self.dt} > radius/4={self.radius/4}")
        if self.seen_margin < 0: raise ValueError("seen_margin")

    def rng(self, *path:int) -> np.random.Generator:
        return make_rng(self.master_seed, *path)

    def manifest(self) -> dict:
        return asdict(self)

def make_rng(master_seed:int, *path:int) -> np.random.Generator:
    ss = np.random.SeedSequence([master_seed, *path])
    return np.random.default_rng(ss)
```

# sim/soup_sim/geometry.py
import numpy as np

def delta(a, b, w, h, boundary):
    d = np.asarray(b, float) - np.asarray(a, float)
    if boundary == "torus":
        size = np.array([w, h])
        d = d - size * np.round(d / size)
    return d

def dist2(a, b, w, h, boundary):
    d = delta(a, b, w, h, boundary)
    return float(np.dot(d, d)) if d.ndim == 1 else np.einsum("...i,...i->...", d, d)

def in_range(a, b, r, w, h, boundary) -> bool:
    return bool(dist2(a, b, w, h, boundary) <= r * r)
```

# append to geometry.py
def contact_interval(pa, va, pb, vb, r, t0, t1, w, h, boundary):
    p0 = delta(pa, pb, w, h, boundary)          # b - a at t0 (minimum image)
    vrel = np.asarray(vb, float) - np.asarray(va, float)
    A = float(np.dot(vrel, vrel)); B = 2*float(np.dot(p0, vrel)); C = float(np.dot(p0, p0)) - r*r
    if A == 0.0:
        if C <= 0.0: return (t0, t1)            # constant separation, already in range
        return None
    disc = B*B - 4*A*C
    if disc <= 0.0: return None                  # never strictly inside
    s = disc**0.5
    lo = t0 + max(0.0, (-B - s)/(2*A))
    hi = t0 + min(t1 - t0, (-B + s)/(2*A))
    return (lo, hi) if hi > lo else None
```
